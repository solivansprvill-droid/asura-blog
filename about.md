# 关于 Asura Blog

Asura Blog 是一个基于 GitHub Pages 的免费静态博客平台。

## 特点

- 📝 **简洁清爽** — 最小化设计，专注内容
- 🚀 **快速加载** — 静态生成，无数据库
- 🔒 **安全可靠** — 由 GitHub Pages 提供支持
- 💰 **完全免费** — 无需付费，无广告
- 🌐 **支持自定义域名** — 使用您自己的域名

## 技术栈

- **静态生成器** — Jekyll
- **托管平台** — GitHub Pages
- **主题** — Minima
