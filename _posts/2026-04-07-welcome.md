# 欢迎来到 Asura Blog

感谢您访问 Asura Blog！这是一个基于 GitHub Pages 的免费静态博客平台。

## 这个博客的特点

### 📝 简洁清爽
- 最小化设计，无多余装饰
- 专注于内容本身
- 响应式布局，适配所有设备

### 🚀 快速可靠
- 静态生成，加载速度快
- 由 GitHub Pages 提供支持
- 99.9% 的可用性保证

### 💰 完全免费
- 无需付费
- 无广告
- 无隐藏费用

### 🌐 支持自定义域名
- 使用自己的域名
- 专业的博客形象
- 完整的 SEO 支持

## 如何使用

### 发布新文章

1. 在 `_posts` 目录下创建新文件
2. 文件名格式：`YYYY-MM-DD-title.md`
3. 添加文章内容和元数据
4. 推送到 GitHub，自动发布

### 文章格式

```markdown
---
layout: post
title: 文章标题
date: 2026-04-07 10:00:00 +0800
categories: [分类1, 分类2]
tags: [标签1, 标签2]
---

# 文章内容

您的文章内容...
```
